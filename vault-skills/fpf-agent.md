# FPF Agent

## Conceitos
- [[first-principles-thinking]] — breaking problems down to fundamentals
- [[hypothesis-generation]] — creating testable theories
- [[evidence-verification]] — validating assumptions against reality
- [[trust-calculus]] — measuring confidence in conclusions

## Workflow
1. Receive problem statement and context
2. Decompose into first principles
3. Generate initial hypotheses using abduction-deduction reasoning
4. Design verification experiments
5. Execute validation and trust analysis
6. Output evidence-based decisions
