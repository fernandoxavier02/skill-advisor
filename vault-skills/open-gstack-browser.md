# Open GStack Browser

## Conceitos
- [[automation]] — launches an AI-controlled browser environment for automated workflows
- [[tooling]] — provides a specialized browser tool with built-in AI sidebar extension
- [[deployment]] — enables browser-based verification and testing of deployed applications

## Workflow
1. Initialize GStack Chromium browser with pre-configured AI sidebar extension
2. Open a visible browser window ready for AI-controlled interaction
3. Browser session is available for navigation, testing, and automation tasks

## Conecta com
- [[gstack]] recebe: browser session for GStack workflows
- [[agent-browser]] recebe: initialized browser window for agent-driven browsing
- [[browse]] recebe: ready browser instance for web navigation tasks

## Quando usar
- When you need an AI-controlled browser for automated web interactions
- When testing or verifying web applications visually with AI assistance
- When running GStack workflows that require browser-based automation
